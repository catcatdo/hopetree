# custom assets

여기에 사용자 커스텀 이미지를 넣어줘.

## 배경
- 경로: `assets/custom/bg/yard.jpg` (또는 `.png`, `.webp`)
- 우선순위: jpg → png → webp

## 고양이
- 경로: `assets/custom/cats/`
- 파일명 예시:
  - `cat01.gif`
  - `cat02.png`
  - `cat03.webp`
  - ...
- 권장: 투명 배경 PNG/GIF, 가로 180~320px

## 참고
- GIF는 `<img>`로 로드하면 애니메이션 그대로 움직여.
